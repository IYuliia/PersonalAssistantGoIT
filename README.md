# Personal Assistant

A command-line personal assistant that helps manage contacts and notes.

## Features

- Contact Management
  - Add, edit, and delete contacts
  - Store contact details (name, phone, email, birthday)
  - Search contacts
  - Birthday reminders
- Note Management
  - Create, edit, and delete notes
  - Add tags to notes
  - Search notes by content or tags
- Data Persistence
  - All data is stored locally
  - Automatic backup

## Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/personal-assistant.git
cd personal-assistant
```

2. Create and activate virtual environment:
```bash
python -m venv env
# On Windows
env\Scripts\activate
# On Unix or MacOS
source env/bin/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run the application:
```bash
python -m personal_assistant
```
